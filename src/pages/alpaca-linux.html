What is Alpaca Linux?

Alpaca Linux is the operating system optimized for Java deployment, emphasizing high performance, security, small size, and flexibility.

  • Protect your Java infrastructure

    Ensure reliable privacy for your runtime and operating system with the full security suite using secure boot support. Receive regular updates for Java and core packages. Prevent both direct and indirect access to a running kernel image. Always stay on Linux Long Term Support.

  • Enjoy the best performance in class

    Use optimal performance & RAM consumption for microservice applications with improved energy efficiency and optimized musl libc. Discover how small and high-performing your containers and images may become.

  • Utilize OS tailored for your workload

    Enjoy the massive variety of Alpaca’s features to deploy modern applications. Utilize the diversity of builds including the standard one, the version optimized for performance, and three additional malloc implementations, that support both GLIBC and MUSL.

Optimized performance

Alpaca’s features include tuned kernel, optimized libc, and optimized malloc options to boost the performance of your applications without sacrificing stability.

Miniature size

With its 2.9 Mb base image size, Alpaca offers the smallest performant docker images, JDK docker images, and native images, making the deployment faster and memory footprint lesser.

Liberica Lite and Liberica NIK support

Liberica Lite, the optimized version of Liberica JDK, enhances the performance and minimizes memory footprint. Liberica NIK allows creating the native images that benefit even more with Alpaca Linux as the foundation.

Enhanced security

The lack of extra components means it is harder to break, and timely, frequent updates reliably remove the vulnerabilities. As a bonus, an additional security hardening is provided by userspace compilation options.

Alpaca Linux Experience

Unified development latform

Liberica JDK and Alpaca Linux provide the stability and consistency you need for development, compatibility with any infrastructure and cloud, and the latest tools and technologies.

Develop with Liberica JDK and Alpaca Linux to create applications in the same environment they will be tested and deployed in.

Stablе future-oriented infrastructure

The best way to build a fail-proof and future-oriented foundation for your architecture is to combine an effective runtime with the OS that runs on every platform.

Together Liberica JDK and Alpaca Linux provide consistency across environments and stability, allowing you to code with future goals in mind.

Container friendly environment

The 2.9 Mb size of the base image, container optimizations, and built-in support for Liberica Lite make Alpaca Linux the go-to solution for the base of containerized applications.

Alpaca Linux is the constantly upgraded ultimate foundation for building secure, lightweight Java containers with low latency, high performance and tiny size.
