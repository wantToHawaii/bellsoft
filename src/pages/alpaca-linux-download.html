Product Description

Alpaca Linux № 3.15.0 is a full-featured operating system allowing an enterprise to run a variety of workloads. It provides flexibility to quickly deploy infrastructure or services with guaranteed support and minimal security vulnerabilities.

The Alpaca Linux Enterprise № 3.15.0 family consists of the following products and extensions:

  • Optimized musl libc library
  • Alternative malloc implementations
  • Optimized Liberica Lite and NIK

Key advantages

  • Security

    Alpaca Linux is built with the goal to enhance the security of your applications and avoid zero-day vulnerabilities. Our implementations for security include networking features, custom build options, kernel hardening, process isolation, and others.

  • Performance

    Performance optimization is ensured by implementing features targeted to deliver reliable and fast stack to run applications for various workloads. These features include kernel optimizations, memory management, and libc implementation optimizations, including our own musl version with reworked APIs.

  • Functionality

    Alpaca Linux is optimized for deploying. To reach this goal it utilizes kernel module compression support to reduce the size of packages in a repository and those installed on a system. The extra modules and firmware packages are separated into lightweight packages and are installed depending on the needs and the target platform.

    Alpaca Linux also fully supports and takes advantage of the BellSoft line of products, including Liberica JDK, Liberica NIK, and Alpaca container images.

More features:

Kernel:

  • Always on Linux LTS (5.10 currently)
  • SecureBoot, signed modules, disabled modules with CVEs
  • Better performance tunables, optimized size (xz)

Userspace based on Alpine Linux aports:

  • Busybox, on-demand coreutils
  • Main/aports & apk package manager
  • No graphics

LIBC:

  • Musl and glibc support
  • Optimized musl, optimized mallocs

Compatibility with Docker, QEMU support:

  • Docker
  • QEMU support

Required downloads

To take full advantage of Alpaca Linux functionality refer to these products from BellSoft:

  • Liberica JDK, the unified Java runtime
  • Liberica NIK, the tool for creating native images
  • Alpaca container images, templates for containers that suit your needs

You can use our configured repository to install these components via CLI. Consult our guide to find out more.

System Requirements

Minimum Linux server system requirements for installation

  • RAM - 512 MB
  • CPU - 1
  • Storage - 2 GB

Recommendations for specific uses

  1. To use the SecureBoot feature, you need the hardware or VM with the UEFI firmware, which is a replacement for the Basic Input/Output System (BIOS) firmware interface. Consult the SecureBoot guide to set up Alpaca Linux for Secure boot.
  2. To install and use the Docker on Alpaca Linux, the minimal system requirements are:
  • RAM - 512 MB
  • CPU - 2
  • Storage - 6 GB

Download and Installation

  1. Download the iso file from this site or use the command line tools to get the file (wget)
  2. Put the iso on the flash drive/DVD/VM
  3. Boot your device with the provided iso
  4. Login with the passwordless root
  5. Execute the “setup-alpaca” script and select the options you need
  6. Perform the final reboot

Consult the advanced installation guide, if needed.
